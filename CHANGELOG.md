# Changelog

## 0.2.0 - 2026-06-26

- Added a dashboard view for tracking multiple packages with local persistence, comma-separated package entry, row refresh actions, and direct navigation back to package measurements.
- Added a result-header version dropdown and Load Latest action for stable version measurements.

## 0.1.0 - 2026-06-26

- Added the package-size CLI and local dashboard for browser-resolved npm artifact measurements.
- Reports raw, gzip, and Brotli byte counts for the resolved `esm.unpkg.com` artifact or metadata response.
- Added shareable dashboard URLs for package specs, targets, conditions, environments, and metadata mode.
- Added local measurement caching for the Node.js API and browser-side cache support for the static dashboard.
- Added stable version history with local size measurements, reload/test actions, trend graphing, npm package links, scope links, and maintainer links.
